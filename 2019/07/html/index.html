Notes on HTML

Jul 2, 2019

HTML stands for Hyper Text Markup Language, and it is a display/mark-up language and NOT a programming language (which requires logical executions)

Any files saved with a .html extension can be opened on a browser. For a website, the index.html file will always be the root page.

Tag

HTML tags normally come in start/end pairs, where a forward slash is added to the end tag. Remnant of XHTML, some tags closes themselves, such as <br> (line break) and <hr> (horizontal line).

Inline / Block

There are two types of tag/element:

  • inline:
    • do not start on a new line
    • take only the necessary width
    • examples: <span>, <img>, <a>
  • block:
    • start a new line
    • take full width available
    • examples: <div>, <h1>, <p>, <form>

Attribute

All tags can have attributes which are placed within the start tag as key/value pairs (id=”000”) and therefore provide additional information.

a

The a tag can attach a link to a content. The attribute href specifies the link location (can be both local and global) and the target attribute can ask the browser to open up a new tab upon clicking (usually for an external site).

ul / ol

The ul tag creates an unordered list, while the ol tag creates an ordered list.

table

form

A form tag allows user interaction/input on the website

img

The alt attribute display the text shown if the image is not found. The width attribute can change the pixels to display. The height attribute can change the height of the photo specified (may be distorted)
